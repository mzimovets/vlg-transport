# Волгоград Транспорт — офлайн-табло трамваев и автобусов

## Что внутри
- `index.html`, `app.js`, `data.js` — всё приложение (без сборки, чистый JS)
- `manifest.json`, `sw.js` — PWA: устанавливается на телефон, работает офлайн
- `icon-*.png` — иконки приложения

## Как запустить локально
Просто открыть `index.html` в браузере — работает без сервера.
Для проверки офлайн-режима (service worker) браузеру нужен http(s), поэтому лучше поднять локальный сервер:
```
cd site
python3 -m http.server 8000
```
и открыть http://localhost:8000

## Как выложить на GitHub Pages
1. Создать новый репозиторий на GitHub (например `vlg-transport`).
2. Положить в него содержимое папки `site/` (можно прямо в корень репозитория).
3. Settings → Pages → Source: **Deploy from a branch**, ветка `main`, папка `/ (root)`.
4. Через 1–2 минуты сайт появится на `https://<ваш-логин>.github.io/vlg-transport/`.
5. На телефоне открыть эту ссылку → в браузере выбрать «Добавить на экран домой» — появится иконка как у обычного приложения, дальше работает офлайн (после первого захода).

Важно: `manifest.json` и `sw.js` рассчитаны на то, что сайт лежит в корне (`start_url` и `scope` — относительные пути `./`), поэтому это будет работать и в подпапке репозитория без правок.

## Ограничения, о которых стоит знать
- **Автобусы**: официальных поминутных расписаний по каждой промежуточной остановке в открытом доступе нет. Время отправления с конечных станций — точное (сверено по агрегаторам данных МУП «ВПАТП №7» и ООО «Волгоградский автобусный парк»), а время на промежуточных остановках рассчитано пропорционально длине маршрута — это оценка, не гарантия.
- **СТ и СТ2**: официальная таблица даёт время только по 5 контрольным пунктам на каждое направление (не по всем 19–21 остановке). Промежуточные остановки сопоставлены этим контрольным точкам приблизительно — точность ниже, чем у обычных трамваев 3 и 4.
- **Геолокация**: список координат остановок (`HUB_COORDS` в `app.js`) — это всего ~16 крупных пересадочных узлов, а не полная база всех ~150 остановок. Если человек не рядом с одной из них, приложение честно скажет, что не нашло ближайшую, и предложит выбрать маршрут вручную. Чтобы расширить покрытие — нужно добавить координаты остальных остановок в этот массив.
